# MyBerry Console

Myberry Console is an example of a visual management tool for [Myberry](https://gitee.com/myberry/myberry) Project.

## Official Site

[myberry.org](https://myberry.org)

## Building

Supports Java 1.8+

```bash
# mvn clean package
```

## Run

```bash
# java -jar myberry-console-*.jar
```
