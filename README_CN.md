# MyBerry 控制台

Myberry 控制台是 [Myberry](https://gitee.com/myberry/myberry) 项目的可视化管理工具示例。

## 官方网站

[myberry.org](https://myberry.org)

## 构建

支持 Java 1.8+

```bash
# mvn clean package
```

## 运行

```bash
# java -jar myberry-console-*.jar
```
